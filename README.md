# پروژه روش‌های عددی - محاسبات علمی

این مخزن شامل پیاده‌سازی سه بخش از پروژه درس **روش‌های عددی** است که با زبان پایتون نوشته شده‌اند.

## محتویات پروژه

### بخش ۱: درون‌یابی
**فایل:** `task1.py`

پیاده‌سازی و مقایسه دو روش درون‌یابی:
- **درون‌یابی لاگرانژ** (Lagrange Interpolation)
- **اسپلاین طبیعی درجه سه** (Natural Cubic Spline) با شرط مرزی طبیعی

**خروجی:**
- ضرایب چندجمله‌ای لاگرانژ
- ضرایب اسپلاین در هر بازه
- نمودار مقایسه‌ای (`interpolation_comparison.png`)

### بخش ۲: تحلیل زمان اجرای الگوریتم‌ها
**فایل:** `task2.py`  
**داده:** `پروژه 3.xlsx`

تحلیل و مقایسه زمان اجرای سه الگوریتم برای اندازه‌های مختلف داده (100KB تا 600KB):

**قابلیت‌ها:**
- خواندن و پاکسازی خودکار داده‌های اکسل
- افزودن خودکار داده برای اندازه 700KB (بخش ب)
- محاسبه میانگین زمان اجرا برای بازه مشخص (بخش ج)
- رسم سه نمودار: میله‌ای، خطی، جعبه‌ای (`part2_charts.png`)

### بخش ۳: انتگرال‌گیری عددی
**فایل:** `task3.py`

محاسبه انتگرال $\int_0^1 e^{x^2} \, dx$ با روش‌های مختلف:
- روش ذوزنقه‌ای (Trapezoidal)
- روش سیمپسون (Simpson's Rule)
- انتگرال گوسی (Gaussian Quadrature)

**خروجی:**
- جدول مقایسه دقت و خطای هر روش
- جدول همگرایی انتگرال گوسی با تعداد نقاط مختلف
